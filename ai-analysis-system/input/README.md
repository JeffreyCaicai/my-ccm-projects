# 📥 Input 文件夹

存放所有**原始资料**，包括：

## 文件类型
- 财经新闻文章
- 手动保存的帖子
- 网页内容摘录
- 研报原文

## 命名规范
```
YYYY-MM-DD-标题.md
```

例如：
- `2026-01-05-央行降准政策解读.md`
- `2026-01-05-新能源汽车行业动态.md`

## 文件模板
```markdown
---
date: 2026-01-05
source: 财联社
category: 宏观政策
---

# 标题

正文内容...
```

## 使用方法
1. 将资料保存为 Markdown 文件
2. 按命名规范命名
3. 运行 `/analyze-news` 命令处理

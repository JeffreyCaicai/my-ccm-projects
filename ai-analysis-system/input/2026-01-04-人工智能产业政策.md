# 人工智能产业政策密集出台 算力需求持续增长

**政策动态：**

近期多部门联合发布《关于加快推进人工智能产业发展的指导意见》，明确提出：

1. 到2027年，AI核心产业规模超过1万亿元
2. 建设一批国家级AI创新中心和算力中心
3. 加大对AI芯片、大模型等核心技术的研发支持

**受益标的：**

**算力基础设施：**
- 中科曙光（603019）：国产高性能计算龙头，服务器市占率领先
- 浪潮信息（000977）：AI服务器出货量快速增长

**AI芯片：**
- 寒武纪（688256）：国内AI芯片领先企业，产品矩阵丰富
- 海光信息（688041）：高端处理器龙头，持续突破

**大模型应用：**
- 科大讯飞（002230）：语音AI龙头，星火大模型持续迭代
- 金山办公（688111）：办公AI应用落地领先

**投资逻辑：**

1. 政策支持力度加大，产业发展确定性强
2. 算力需求持续增长，基础设施建设加速
3. 大模型应用场景不断拓展，商业化进程提速

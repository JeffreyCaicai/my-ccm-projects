# 新能源汽车销量创新高 产业链迎来黄金期

**行业数据：**

2025年中国新能源汽车销量突破1500万辆，同比增长35%，渗透率超过40%。

**重点公司表现：**

1. **比亚迪（002594）**：全年销量超过350万辆，连续三年蝉联全球新能源汽车销量冠军
2. **宁德时代（300750）**：动力电池市占率稳居全球第一，扩产计划加速推进
3. **特斯拉上海工厂**：产能持续提升，出口量大幅增长

**上下游分析：**

- 锂电池：需求旺盛，头部企业产能利用率高位运行
- 充电桩：政策支持力度加大，市场空间广阔
- 汽车芯片：国产替代加速，相关企业受益明显

**投资机会：**

1. 整车龙头：比亚迪（002594）、长城汽车（601238）
2. 电池产业链：宁德时代（300750）、亿纬锂能（300014）
3. 充电设施：特锐德（300001）

**风险提示：**

- 补贴退坡可能影响短期销量
- 原材料价格波动风险
- 市场竞争加剧

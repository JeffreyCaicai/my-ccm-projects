# 央行宣布降准0.5个百分点 释放流动性超万亿

**核心要点：**

1. 中国人民银行今日宣布，决定于2026年1月15日下调金融机构存款准备金率0.5个百分点。

2. 此次降准将释放长期资金约1.2万亿元，进一步支持实体经济发展。

3. 分析师普遍认为，此举将利好银行股（600036 招商银行、601398 工商银行）和券商板块。

**市场影响：**

- 银行板块：降准降低银行融资成本，提升净息差预期
- 券商板块：流动性宽松利好股市，券商业绩有望改善
- 地产板块：宽松政策有望缓解房企资金压力

**投资建议：**

结合当前经济环境，建议关注以下方向：
1. 大型国有银行（工商银行601398、建设银行601939）
2. 头部券商（中信证券600030）
3. 优质地产股

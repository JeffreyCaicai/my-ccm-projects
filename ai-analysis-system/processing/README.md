# ⚙️ Processing 文件夹

存放所有 **AI 处理结果**，等待用户消化。

## 文件类型
- 每日新闻分析报告
- 周报/月报
- 股票筛选结果
- 行业分析总结

## 自动生成的文件
| 文件名格式 | 说明 |
|-----------|------|
| `YYYY-MM-DD-新闻分析.md` | 每日新闻分析 |
| `YYYY-WXX-周报.md` | 每周总结 |
| `YYYY-MM-月报.md` | 每月总结 |
| `YYYY-MM-DD-股票筛选.md` | 股票推荐列表 |

## 使用流程
1. AI 自动生成分析报告到此目录
2. 用户阅读并消化内容
3. 有价值的内容运行 `/extract-insights` 移至 output
